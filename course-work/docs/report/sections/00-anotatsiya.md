# АНОТАЦІЯ

У курсовій роботі розглянуто алгоритм бітонічного сортування як представника сортувальних мереж із регулярною структурою compare-swap операцій, зручною для розпаралелювання на багатоядерних обчислювальних системах. Метою дослідження є розробка послідовної та паралельної реалізацій алгоритму мовою програмування Python з використанням процесів і спільної пам'яті (`multiprocessing`, `multiprocessing.shared_memory`) та експериментальне підтвердження практичного прискорення не менше ніж у 1,2 рази порівняно з послідовною версією за достатньо великих обсягів вхідних даних.

У роботі систематизовано теоретичні відомості про бітонічне сортування та відомі підходи до паралельної реалізації; наведено псевдокод ітеративного варіанту алгоритму. Розроблено послідовну реалізацію на основі ітеративної сортувальної мережі з доповненням масиву до найближчого степеня двійки; досліджено залежність часу виконання від розміру масиву. Обґрунтовано вибір засобів паралелізму в Python і спроєктовано паралельний алгоритм як послідовність фаз, що відповідають шарам мережі, з бар'єрною синхронізацією між кроками. Реалізовано паралельну версію зі статичним розподілом діапазонів індексів між процесами та спільним буфером даних. Проведено тестування коректності обох реалізацій, зокрема порівняння результатів паралельного та послідовного сортування на різних наборах даних і конфігураціях процесів.

Результати експериментів показали, що для малих масивів переважають накладні витрати на організацію паралельних обчислень, тоді як починаючи приблизно з 16 384 елементів паралельна реалізація стабільно перевершує послідовну. На найбільшому дослідженому масиві (4 194 304 елементи) зафіксовано максимальне прискорення 4,642 при восьми процесах; найвища ефективність (0,935) спостерігалася при двох процесах. Отримані дані підтверджують досягнення цільового прискорення понад 1,2 для великих обсягів даних і доцільність застосування запропонованого підходу до паралелізації бітонічного сортування в середовищі Python.

Обсяг роботи: пояснювальна записка (вступ, п'ять розділів, висновки, список використаних джерел), **6 таблиць**, **11 рисунків**, **3 додатки** (листинги програмних модулів реалізації, тестів і бенчмарку).

**Ключові слова:** бітонічне сортування, сортувальна мережа, паралельні обчислення, Python, `multiprocessing`, спільна пам'ять, прискорення, ефективність, тестування, експеримент.
